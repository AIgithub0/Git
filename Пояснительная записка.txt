Пояснительная записка
2.1 Темой проекта является тренажер сурдоперевода
2.2 Назначаение данного приложение, это научить людей разговору с помощью жестов, применятеся данное приложение среди людей занимающихся сурдопереводу или также для глухонемых людей
2.3 Задача проекта, создать приложение где пользователь может пройти обучение и также проверить себя с помощью тестов
Было использовано математический метод такой, как высчитывание процента решенных тестов (проблем с этим не было)
Алгоритм задачи проекта
1) Создать файлы для теста и обучения (С файлов считываются изображения и выводятся пользователю по нажатию клавиши и также кнопки в тестах)
2) Также работа с другой программой (такой как SQL studio), это нужно было для для записи процента выполненых правильных тестов в таблицу базы данных 
2.4 На приложение будет большой спрос так как его будет использовать средний круг лиц, куда входят инвалиды и люди работающие с ними. 
2.5 Для разработки данного приложение, я использовал учебники Яндекс лицея, в частности учебник по базе данных, и по csv файлам, также был использован интернет для нахождения изображение (руки с жестами)
